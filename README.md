# AdvanceCraft II

This is a project that is half Advance Wars, half Starcraft II. It is being
written for fun to learn the Bevy Engine.

# Installation

The project requires Rust and Cargo to be installed.

It can be run with

```sh
cargo run
```

or you can create a binary with

```sh
cargo build
```
